# Console Night Watch (FNAF-Inspired)
THIS IS A DEMO ITS BAD

Single-file browser-console survival demo with:
- visible animated enemy movement through rooms
- animated door-window encounters when enemies reach either hallway door
- pseudo-3D camera rendering on canvas (perspective floor, depth, CRT/static effects)
- power / camera / door / light management from 12 AM to 6 AM

## Run from Chrome DevTools

1. Open any webpage.
2. Open DevTools.
3. Paste the full contents of `fnaf-inspired-console.js` into Console.
4. Press Enter.

## Controls

- **Open Cameras / Close Cameras**
- **Door / Light** controls on left and right panels
- **Restart** after win/loss
- **Close** to remove overlay

## Notes

This is an original implementation inspired by office-survival mechanics and is not a 1:1 recreation of any copyrighted game.
